login-predictor-frontend: React ile yapılmış kullanıcı arayüzü
Backend: PHP ile yazılmış API, tahmin algoritmalarını çalıştırır
canlı=>https://case1-nu.vercel.app/

# Teknolojiler

- **Frontend**: React.js, Axios
- **Backend**: PHP

- 
Gerekli Yazılımlar

Node.js (v14 veya üstü)

npm (Node.js ile birlikte gelir)

PHP (Backend için)
Frontend Kurulumu
git clone https://github.com/HsnEmre/Case.git
cd Frontend
npm install
npm start
Backend Kurulumu
cd Backend
composer install
php -S localhost:8000
projeyi http://localhost:8000/predictions.php tarayıcınızdan açarak çalıştırabilirsiniz
- Bu projeyi çalıştırabilmek için aşağıdaki yazılımların bilgisayarınızda yüklü olması gerekmektedir:
- [Node.js](https://nodejs.org/) (v14 veya üstü)
- npm (Node.js ile birlikte gelir)
